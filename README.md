### Binary Trees in Rust — BST, AVL, Red-Black Tree

Implementation of classical binary tree data structures in safe Rust.

- Implemented BST, AVL, and Red-Black Tree structures;
- supports insertion, search, removal, traversal, rotations, and balancing logic;
- uses ownership-aware modeling with `Box` and index-based links;
- includes unit tests and CLI tools for interaction and visualization;
- built without `unsafe`.

Why this project matters:

- it demonstrates deeper interest in correctness, memory, and data structures;
- it strengthens the fundamentals behind backend development;
- it shows technical curiosity beyond framework usage.

[View project](https://github.com/amauribsjr/binary-trees-rust)
